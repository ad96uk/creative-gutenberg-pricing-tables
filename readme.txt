== Description ==
TrendyWP Pricing Table Library is a WordPress plugin designed to provide a rich collection of modern, stylish pricing table templates. Effortlessly create and customize responsive pricing tables that integrate with the Gutenberg editor and enhance your websiteâ€™s user experience.

== Features ==
- **Extensive Template Library**: Access a wide range of trendy and professional pricing table designs.
- **Highly Customizable**: Tailor colors, fonts, and layouts to match your brand's aesthetics.
- **Responsive Design**: Ensure your pricing tables look great on any device.
- **User-Friendly Interface**: Easily create and manage pricing tables with an intuitive interface.
- **Gutenberg Compatible**: Integrates smoothly with the WordPress Gutenberg editor.
- **Free to Use**: Enjoy all core features without any cost.

== Installation ==
1. Download the `TrendyWP Pricing Table Library` plugin.
2. Upload the plugin files to the `/wp-content/plugins/` directory or install via the WordPress Plugin Directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Start creating and customizing your pricing tables from the plugin's menu in your WordPress dashboard.

== Usage ==
1. After activation, navigate to the "TrendyWP Pricing Table Library" menu in your WordPress dashboard.
2. Browse and select a template from the library.
3. Customize the appearance, content, and layout using the provided options.
4. Embed your table into any page or post using the shortcode provided.
